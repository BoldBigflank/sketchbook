# Sketchbook

Create sketches. I'll add functionality to the sketchpad project as I go.

## Install

`npm install`
`npm build`

## Run

`npm start`

# Planned Features

* Save image
* Save json
* Load json
* Export as large image
* View saved sketches (Gallery)
* Deploy to GH Pages
